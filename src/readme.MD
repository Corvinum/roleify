# ROLEIFY

## ¿Qué es Roleify?
Es un sistema de gestion y herramientas para rol. 
Actualmente se encarga de guardar personajes.
 
## Caracteristicas
- Menu principal con distintos submenus.
- Control de errores
- Coloreador de mensajes
- Una base de datos(entre comillas)
  - Gestión
    - Personaje.
      - Creación.
      - Galería.
      - Modificación.
      - Eliminación.
